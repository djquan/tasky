# PWA Icons

Add the following icons to this directory for the PWA:

- `icon-192.png` - 192x192px icon
- `icon-512.png` - 512x512px icon
- `favicon.ico` - Browser favicon

Until you add these, the PWA will work but won't have proper icons for installation.

You can generate these from a single SVG or PNG using tools like:
- https://realfavicongenerator.net/
- https://favicon.io/
